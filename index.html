    Ajax and JSON

    The web app uses AJAX to read data from the JSON file. However, I could not send data to the JSON file using the same methods. The JSON was hardcoded and I could not remotely send data to it without a back-end server. When I tried to sending data using AJAX I got error “405 Method Not Allowed” due to certain restrictions by the browser and possibly other things. I have still added “Create” and “Delete” functionality though. It’s just that they work on the front end only and, on page refresh all changes made will disappear.
